# International Research Opportunities

A curated list of research internships and programs for international students, particularly in CS, tech, math, and bio. Started this as I was looking for opportunities myself and figured it'd help others too.

Open to contributions! If you know of programs that accept international students, feel free to submit a PR.

## Contributing

Found a program that accepts international students? Please submit a PR with:
- Program name and link
- Institution and location
- Fields of study
- Typical deadline
- Any important notes (funding, duration, eligibility)

## Notes

- Deadlines are approximate and vary year to year so, always check official websites
- Most programs require you to be currently enrolled as an undergraduate
- Some programs may have citizenship restrictions despite accepting international students, read carefully
